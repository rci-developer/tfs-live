Hide product price until the correct password is entered or until login.

== Description ==

Hide Price of the products on shop and product detail page until user is not logged in or until password is entered.

[youtube https://www.youtube.com/watch?v=2ltqqUxkx5I]

= Main features =
* **Hides the product's price until the correct password is entered**
* **Hides the product's price for specific users**
* **Hides the product's price until the user is logged in**
* **Hides price from shop page and product detail page as well**
* **Hides the product's price for none**
* At shop page only the details of a prodcut is shown, either it is a  simple product.
* Full customizable, most of the things are dynamic and can be customized by admin.
* Enable/Disable all the features of plugin
* Efficient for valid user only.
* Easy to use and install
* Easy to configure
* Show/Hide password field.
* **Excellent language support**

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Hide Price Until Login, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Hide Price Until Login" and click Search Plugins. Once you've found our Hide Price Until Login plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Hide Price Until Login plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
